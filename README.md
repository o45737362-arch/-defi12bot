# Telegram Website Uptime Bot

A Telegram bot that monitors websites and alerts you when they go down or come back up.

## Features
- 🔍 Monitor multiple websites
- 📨 Instant Telegram notifications
- ⏱ Track response times
- 🔔 Customizable notifications (down/up alerts)
- 📊 Real-time status checking
- 🚀 Easy deployment on Railway

## Commands
- `/add <url>` - Add website to monitor
- `/remove <url>` - Remove website from monitoring
- `/list` - Show all monitored URLs
- `/status` - Check current status
- `/pingnow` - Force immediate check
- `/settings` - Configure notifications
